Week 2 - Prompts + Variables

Prompt: A "prompt" is a way to interact with users in a programming environment, typically in web development or scripting. It's a dialog box or input field that allows users to enter data, which is then captured by the program. The program can use the data entered by the user to perform various tasks or make decisions. In JavaScript, for example, you can use the prompt() function to display a prompt and collect user input.

Variables (Vars): Variables are used to store and manage data in programming. They act as containers or placeholders that can hold different types of values, such as numbers, text, or objects. Variables provide a way to reference and manipulate data within a program. In most programming languages, you need to declare a variable before using it, specifying its name and optionally its initial value.
